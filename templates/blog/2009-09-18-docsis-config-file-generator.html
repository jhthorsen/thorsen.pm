Online DOCSIS config file editor

18th of September, 2009

I’ve written an online DOCSIS config file editor which (of course) can be used to generate config files for equipment that follow the DOCSIS specification.

The frontend is written using Catalyst, and the “backend” use DOCSIS::ConfigFile, meaning the whole thing is powered by Perl.

(The frontend will soon support “shared secret”, which the backend already supports)
